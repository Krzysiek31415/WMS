# WMS

A simple Warehouse Management System (WMS) project in C++.

At this stage, the project focuses on:

* warehouse model (products, tasks, employees)
* task assignment logic
* queues and a simple scheduler
* simulation of system behavior (without UI and API)

Project structure:

* `core` – system logic
* `simulator` – environment for running scenarios
* `tests` – tests (work in progress)

The project is developed iteratively – starting with core and simulation, then expanding with additional modules.
